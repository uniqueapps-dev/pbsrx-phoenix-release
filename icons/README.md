# PBSRx Icons

Place the following PNG files in this directory:

- `icon-192.png` — 192×192 pixels — used for Android home screen installation
- `icon-512.png` — 512×512 pixels — used for splash screen and PWA store listings

Design guidelines:
- Background: #0a1e3d (PBSRx navy)
- Foreground: #d97706 (PBSRx amber) showing "Rx" or pill icon
- Maskable safe zone: inner 80% of canvas
